# Animated Social Profile Card

A modern animated social profile card built using HTML and CSS.

## Features

- Glassmorphism UI
- 3D Flip Animation
- Responsive Design
- Social Media Icons
- Smooth Hover Effects
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## Preview

Front side contains:
- Profile Image
- Name
- Profession

Back side contains:
- Instagram
- X (Twitter)
- GitHub
- LinkedIn
